# NeuroLearn

## Description

NeuroLearn is an AI-powered learning platform that helps students learn new topics through personalized learning roadmaps and AI-generated video explanations.

## Features

- **Personalized Learning Roadmaps**: Generate custom learning roadmaps for any topic
- **Youtube Video Explanations**: Get Youtube video explanations for each subtopic
- **User Authentication**: Secure user registration and login
- **Roadmap Management**: Create, view, and manage learning roadmaps

## Tech Stack

- **Frontend**: React, Redux, Tailwind CSS
- **Backend**: Node.js, Express, MongoDB
- **AI**: Google Gemini API

## Usage

1. Open the app in your browser: http://localhost:3000
2. Sign up or log in to your account
3. Create a learning roadmap for any topic
4. Watch AI-Searched video explanations for each subtopic
